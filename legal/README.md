# Legal & App Store bundle

| File | Use |
|------|-----|
| `EULA.md` | End-user license — host publicly and link from app / onboarding. |
| `PRIVACY_POLICY.md` | Privacy policy — required for App Store; wire URLs in `app.config.js` extras. |
| `DEMO_ACCOUNT_FOR_APPLE.txt` | Reviewer demo account instructions. |
| `AGE_RATING_AND_SCREENSHOTS.txt` | Store metadata notes. |

Replace `example.com` URLs in templates with your production legal pages before submission.
